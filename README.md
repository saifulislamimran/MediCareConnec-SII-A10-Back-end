## 🔥 Architecture & Security Highlights

This backend is built on the principle of **"Never Trust the Client"**, implementing global standard security measures:

*   **🛡️ Server-Side Price Authority:** Client-side price manipulation is completely neutralized. Stripe Payment Intents are generated by querying the exact `consultationFee` directly from the MongoDB Doctor schema.
*   **🚫 Concurrency & Overbooking Control:** Strict time-slot validation logic prevents multiple patients from booking the exact same slot for a specific doctor.
*   **🔒 Bulletproof RBAC Middleware:** Integrated `verifyToken` and `verifyRole` middlewares intercept every request. Admin, Doctor, and Patient routes are completely isolated at the server level.
*   **📦 Server-Side Pagination & Filtering:** Prevents data hoarding and browser crashes by utilizing Mongoose `.skip()`, `.limit()`, and `$regex` for dynamic search and pagination on the backend.
*   **🌱 Automated Seeding Protocol:** Features a secure endpoint to automatically inject realistic, professional doctor profiles into an empty database for rapid deployment testing.
